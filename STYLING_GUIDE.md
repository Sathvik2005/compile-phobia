# Compile Phobia Styling Guide

## Design System

### Color Palette
- **Surface**: #fff9f2 (light background)
- **Ink**: #182028 (dark text)
- **Accent**: #0f766e (teal primary)
- **Success**: #10b981 (green)
- **Warning**: #d97706 (amber)
- **Danger**: #b42318 (red)

### Layout System
- **Hero Section**: 40px padding, 24px border-radius
- **Grid Layout**: 4-column for stats, responsive to mobile (480px breakpoint)
- **Max Width**: 1200px for content containers

### Components
- Pie Chart: Language distribution visualization
- Bar Chart: Contributor timeline (top 10)
- Score Circle: 0-100 quality score with color coding
- Recommendations Grid: Flexible card layout
- Comparison Grid: Side-by-side metric comparison

### Responsive Breakpoints
- 1024px: Tablet layout
- 768px: Mobile layout
- 480px: Small phone layout

### Animations
- `fadeIn`: 0.3s fade-in effect for components
- `slideDown`: 0.3s slide-down for modals
- `pulse`: Continuous pulse for skeleton loaders
